Information Literacy Project

Project Title Project Annotation Project Download(s)

Intro to News Literacy: Misinformation, Disinformation, and Evaluating a News Story

 This is an instruction session created as the final project for LIS 60618, Information Literacy Instruction. This project addresses the AASL's "Curate" frame and "Create" competency, which states that students must be able to assess the validity and accuracy of information, an increasingly important topic in today's information landscape and one which is integral to information literacy teaching. It involved a four-step planning process for a 20-minute lesson which was then taught to classmates virtually. This project represents the first experience teaching and receiving feedback, which will be invaluable in my future as a teacher librarian.

Pottermore for Muggles

 This website was created as a final group project for LIS 60675, Reference Services and Sources to Youth. This demonstrates the ability to use Weebly to design and host websites as well as the ability to collaborate with others on projects. It follows the [Ohio Guideline] for reference services. My primary contribution was the Implications page, which tied the site's content to practice in libraries.

Books Unlimited Website

 This website was created as the final project for the Digital Technologies course, LIS 60511, Internet Fundamentals, and is hosted through the student project servers provided by Kent State. This project required the use of HTML, CSS, and PHP to create a dynamic website that simulates some of the functions of an electronic storefront. I used SSH to upload the files to the server. These skills can also be used to manage library services and technologies in the future. It allowed me to build the skills to be able to compile this electronic portfolio, as well.
